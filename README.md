# dvd_manager

Recreating a DVD collection from CS 401

Java Files
• DVD.java - A class to model a single DVD, including its title, rating and total running
time.
• DVDCollection.java - A class to model a collection of DVDs using an array.
• DVDUserInterface.java - An interface that describes the operation required for any user
interface to this DVD collection.
• DVDGUI.java - A class that implements the DVDUserInterface interface and provides a
graphical user interface to the DVD collection.
• DVDConsoleUI.java - A class that implements the DVDUserInterface interface and
provides a console user interface to the DVD collection.
• DVDManager.java - A class that contains a main method that launches one of the two
user interfaces for the user to work with the DVD collection based on the user input.
